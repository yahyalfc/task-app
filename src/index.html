This is a task manager application


In order to create a user: POST name, email, password {{url}}/users

Log in: POST email, password {{url}}/login

Read profile: GET {{url}}/users/me

Update profile: PATCH {{url}}/users/me

Delete user: DELETE {{url}}/users/me

Upload avatar: POST {{url}}/users/me/avatar

Get avatar: GET {{url}}/users/avatar/id

Delete avatar: DELETE {{url}}/users/me/avatar

Logout: POST {{url}}/users/logout

Logout from all devices: POST {{url}}/users/logoutall


In order to create a task: POST description, status {{url}}/tasks

Read tasks: GET {{url}}/tasks

Read tasks in descending order: GET {{url}}/tasks?sortBy=createdAt:desc

Delete task: DELETE {{url}}/tasks/id

Update Task: PATCH {{url}}/tasks/id
